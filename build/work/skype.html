Garamondieu Mac Skype Theme

what is it

Garamondieu is a chat Theme for the mac version of skype. It was created for a competition organized by skype. It's made to look like an antique printed book, the title referencing Claude Garamond.

what I Did

I designed and built it. The chat area in the skype app is really a web view, so the theme is built purely on HTML/CSS.
